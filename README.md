# Queens LinkedIn Solver

Tugas Kecil 1 IF2211 Strategi Algoritma.

Queens (game pada platform LinkedIn) adalah permainan logika di mana pemain harus menempatkan N buah ratu pada papan berwarna dengan ketentuan:

- Setiap baris hanya boleh berisi tepat satu ratu.
- Setiap kolom hanya boleh berisi tepat satu ratu.
- Setiap wilayah warna (region) hanya boleh berisi tepat satu ratu.
- Tidak ada dua ratu yang boleh saling bertetangga, termasuk secara diagonal.
  Jumlah ratu yang ditempatkan sama dengan jumlah baris, kolom, dan wilayah warna (N).

Program Queens LinkedIn Solver bertujuan untuk menemukan satu solusi penempatan ratu yang valid pada papan berwarna yang diberikan, atau menyatakan bahwa tidak ada solusi yang memenuhi seluruh constraint Pendekatan yang digunakan adalah algoritma brute force. Program diimplementasikan sebagai aplikasi desktop dengan dukungan input dan output dalam format gambar serta berkas teks (.txt).

## Fitur

- Antarmuka pengguna interaktif dengan pembaruan proses brute force secara langsung (live update).
- Input papan warna secara manual melalui UI.
- Ekspor hasil penyelesaian dalam format gambar atau teks.
